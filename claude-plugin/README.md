# Lumi Inline (Claude Code plugin)

Lumi teaches you new tech concepts as you use Claude Code — one short, friendly lesson the
first time each concept shows up. Works in the terminal, desktop, and **the mobile app**,
using your existing Claude subscription (no API key).

## Install

```
/plugin marketplace add stefanbogdanmda/lumi
/plugin install lumi@lumi
```

Then just use Claude Code normally. When something new comes up, Lumi adds a "🪄 Lumi — quick
lesson" at the end of the reply.
